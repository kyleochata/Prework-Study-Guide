HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Add CSS.file: link: rel="stylesheet" href="location of css.file"
  • Add script: scrip src="file location"

CSS

  • Margin indicates how much space we want around the outside of an element
  • Padding is the space we want around the content within an element. (Gap between border and content)
  • Class selectors: .name
  • ID selector: #name

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commit all files in the working branch
  • git commit -m "description": flag an associated message with the commit
  • git pull origin main: pull base branch into feature branch. Best practice to do before pushing any code

JavaScript

  • console.log(): Used to see the output of a function or code within the console terminal
  • If statements: conditional statements that can interrupt control flow
  • Array: single variable used to hold a group of data (strings [must have " "], numbers, Boolean value[true/false with no " "])
  • for loop: will run a task on an array by allowing a single block of code to be executed over and over. for (statement 1; statement 2; statement 3;) { code block}
    • First statement: starting point of loop. Declare a variable and it's starting point
    • Second statement: Condition. As long as variable in statement 1 is less than the array, the loop will run. x < variable_name.legnth
      • Note: the variable_name was previously defined outside of the for loop
    • Final Statement: allows array to iterate over each item. Since it increases by 1 each time the loop is run: x+1 or x++
  • function: block of code designed to performa specific task
    • function name(parameter1, parameter2, etc...) { //code to be executed}
  • if: specify block of code to be executed, if a specified condition is true
  • else: code to be executed if the specified condition is not true
  • else if: specify a new condition to test, if the first condition is false
  • Math.floor(Math.random() * [(array.length) or (max - min + 1)]): generate a random integer between 2 set points
