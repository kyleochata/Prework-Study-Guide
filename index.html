Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Add CSS.file: link: rel="stylesheet" href="location of css.file"

CSS

  • Margin indicates how much space we want around the outside of an element
  • Padding is the space we want around the content within an element. (Gap between border and content)
  • Class selectors: .name
  • ID selector: #name

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commit all files in the working branch
  • git commit -m "description": flag an associated message with the commit
  • git pull origin main: pull base branch into feature branch. Best practice to do before pushing any code
